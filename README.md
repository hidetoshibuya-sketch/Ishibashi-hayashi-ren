# お囃子ノーツ（PWA）

縦スクロールの太鼓稽古アプリです。左右2レーンに音玉が上から落ちてきて、中央には歌（テケ・トコ等）のブロックが流れます。
青い玉＝左手、赤い玉＝右手。判定ラインに重なった瞬間に対応する手で叩くと玉が消えます。採点はありません。
譜面はアプリ内の編集画面で、玉や言葉をドラッグして直感的に調整でき、JSONとして書き出せます。
オフライン動作・ホーム画面追加に対応しています。

## ファイル構成
- `index.html` … アプリ本体
- `manifest.webmanifest` … PWA設定
- `sw.js` … オフライン用 Service Worker
- `icons/` … アプリアイコン
- `scores/auto-from-audio.json` … 録音から自動検出した打点（参考データ）
- `scores/sample-matsuri.json` … サンプル

## 動かし方
PWA（オフライン・ホーム画面追加）を使うには **http/https 配信が必要**です。ダブルクリック（file://）でも音と画面は動きますが、PWA機能は無効です。

ローカル確認（このフォルダで）:
```
python3 -m http.server 8000
```
→ ブラウザで `http://localhost:8000/`。スマホは同じWi-FiでPCのIPにアクセス。ブラウザメニューの「ホーム画面に追加」でアプリ化できます。

※ 更新が反映されないとき：以前のバージョンを開いていた場合、ブラウザのキャッシュが残っていることがあります。一度だけ「強制再読み込み（ハードリロード）」するか、サイトデータを消すと最新になります。今回からネットワーク優先の更新に変更したので、以降はオンラインなら常に最新が読み込まれます。

## 操作
- 青い玉＝左手、赤い玉＝右手。玉が判定マルに重なった瞬間に「左手／右手」ボタン（または画面の左半分／右半分、キーボード F・J）で叩くと玉が消えます。
- 中央の言葉ブロックは歌のガイド（叩く対象ではありません）。
- タイミングが外れても音は鳴ります（採点なし）。判定は甘め（前後約0.55秒）にしてあり、玉は消えやすくなっています。

## 速さ調整
スタート画面と編集画面の両方に「速さ」スライダーがあります。玉が上端から判定ラインに届くまでの秒数で、小さいほど速くなります。変更は即反映され、自動保存されます。

## 編集画面（ドラッグで調整・自動保存）
スタート画面の「✎ 譜面を編集」を開きます。縦のレール上に、左手玉・中央の言葉・右手玉が時刻順に並びます。**編集内容は自動保存され、アプリを閉じても残ります**（同じブラウザ内）。
- **上下にドラッグ** … 玉や言葉の時刻（間隔）を調整。0.05秒刻みでスナップ。
- **玉を左右にドラッグ** … 左手玉／右手玉を入れ替え。
- **マグネット整列** … 玉と言葉を近く（約0.13秒以内）に置くと自動で同じ時刻に吸着し、横一列にそろいます。
- **言葉ブロックをタップ** … 文字を自由に変更。
- **テンプレ単語** … ア／テン／テレ／テ／ケ／スク／トコ／テットン／トントコ／ステスク／ス のボタンを押すと、その言葉ブロックを中央に追加。自由入力欄＋「＋言葉」も使えます。
- **× ボタン** … その玉／言葉を削除。
- **＋左手玉／＋右手玉** … 画面中央の時刻に玉を追加。
- **区間コピー（繰り返し）** … 黄色い帯の「始／終」の取っ手で区間を選び、「⧉ 後ろに複製」でその区間の玉と言葉をすぐ後ろにコピー。**帯の本体（取っ手以外）を上下にドラッグすると、区間内の玉と言葉をまとめて移動**できます（複製した塊の位置調整に便利）。「区間を削除」でまとめて消去。
- **速さ／拡大** … 落下の速さと、レールの拡大率。
- **▶ 試す** … 今の内容ですぐプレイ。**JSON保存** … `.json` でダウンロード（バックアップや共有用。次回「JSONを読み込む」で復元）。

## 譜面JSON（書き出される形式）
```json
{
  "title": "曲名",
  "scrollTime": 2.6,
  "notes": [
    { "time": 1.79, "hand": "L" },
    { "time": 2.10, "hand": "R" }
  ],
  "words": [
    { "time": 1.79, "text": "テケ" }
  ],
  "_v": 2
}
```

- `notes[].time` … 叩く時刻（秒）。`hand` … `"L"`（左手・青）か `"R"`（右手・赤）。
- `words[].time` … 表示時刻（秒）。`text` … 中央に流れる言葉。
- `scrollTime` … 落下の速さ（小さいほど速い）。
- 旧形式（`notes` に `text`/`balls`）のJSONを読み込むと、自動で左右の玉＋中央の言葉に変換されます。

## 別の録音から作り直すには
新しいMP3を解析して打点を起こす場合は、その音声を送ってください。タイミングを再検出して譜面の下書きを作り直します。
